# Project

`flyto-docs` is the public technical documentation site for Flyto2.

Primary jobs:

- Explain how Flyto2 works across the five product lines.
- Document `flyto-core` modules and runtime primitives.
- Provide security warroom, CTEM, BYO integration, scoring, and evidence
  mechanics.
- Provide citation-ready technical pages for AI search and answer engines.

Site role:

- `flyto2.com` is commercial positioning.
- `blog.flyto2.com` is educational search-intent content.
- `docs.flyto2.com` is technical mechanism, architecture, and reference.
