# Roadmap

## P0 before production release

- Keep `public/robots.txt`, `public/llms.txt`, and `public/llms-full.txt`
  aligned with public docs navigation.
- Run `npm run docs:build`.
- Include `flyto2-product-gate` output in the release packet.
- Keep Core references and all 15 module locale mirrors reproducible from their
  public source repositories.

## P1 before controlled production

- Add Cloud / Apps / Automation docs equal in quality to Security docs.
- Add enterprise, airgap, open-core, billing/entitlement, and audit-log docs.
- Translate high-value onboarding and Core narrative pages beyond the generated
  module reference; advertise only translations that actually exist.

## P2

- Add Data, Zero-person Agent, and Big Data / Intelligence design notes as the
  future repos mature.
- Evaluate a hosted search service only if local per-locale indexes no longer
  meet latency or bundle budgets.
