# VitePress Runtime

This directory owns the Flyto2 Docs build-time and browser-time site runtime:
locale routing, navigation, source-backed sidebars, metadata, structured data,
sitemaps, alternate-language links, search, theme components, and styles.

See [Architecture](../ARCHITECTURE.md) for the publication flow and the
[generated Docs code reference](../reference/docs-code.md) for every maintained
function and configuration method. Run `npm run verify` after changing this
directory.
