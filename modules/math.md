# Math

Mathematical operation modules for numeric computations.

## Modules

| Module | Description |
|--------|-------------|
| `math.abs` | Absolute value |
| `math.ceil` | Round up to nearest integer |
| `math.floor` | Round down to nearest integer |
| `math.power` | Raise to power (exponentiation) |
| `math.round` | Round to nearest integer or decimal places |
