# Document

Modules for working with office documents. Read, write, and convert between Excel, PDF, and Word formats.

## Modules

| Module | Description |
|--------|-------------|
| `excel.read` | Read data from Excel files |
| `excel.write` | Write data to Excel files |
| `pdf.fill_form` | Fill PDF form fields |
| `pdf.generate` | Generate PDF documents |
| `pdf.parse` | Parse and extract PDF content |
| `pdf.to_word` | Convert PDF to Word document |
| `word.parse` | Parse Word document content |
| `word.to_pdf` | Convert Word document to PDF |
