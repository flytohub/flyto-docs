# Array Operations

Modules for manipulating arrays. Transform, filter, combine, and reshape array data within workflows.

## Modules

| Module | Description |
|--------|-------------|
| `array.chunk` | Split array into chunks |
| `array.compact` | Remove null/empty values |
| `array.difference` | Find elements in first array not in others |
| `array.drop` | Drop first N elements |
| `array.flatten` | Flatten nested arrays |
| `array.group_by` | Group elements by a key |
| `array.intersection` | Find common elements |
| `array.join` | Join elements into string |
| `array.map` | Transform each element |
| `array.reduce` | Reduce to single value |
| `array.take` | Take first N elements |
| `array.zip` | Combine arrays element-wise |
