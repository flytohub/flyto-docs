# Logic

Boolean logic modules for building conditional expressions in workflows.

## Modules

| Module | Description |
|--------|-------------|
| `logic.and` | Logical AND of multiple conditions |
| `logic.contains` | Check if value contains substring or element |
| `logic.equals` | Check equality between values |
| `logic.not` | Logical NOT (invert boolean) |
| `logic.or` | Logical OR of multiple conditions |
