# Kubernetes

Modules for interacting with Kubernetes clusters. Manage pods, deployments, and resources.

## Modules

| Module | Description |
|--------|-------------|
| `k8s.apply` | Apply Kubernetes manifest |
| `k8s.describe` | Describe Kubernetes resource |
| `k8s.get_pods` | List pods in namespace |
| `k8s.logs` | View pod logs |
| `k8s.scale` | Scale deployment replicas |
