# Image Processing

Modules for image manipulation, conversion, and analysis. Supports resizing, cropping, compression, OCR, QR code generation, and watermarking.

## Modules

| Module | Description |
|--------|-------------|
| `image.compress` | Compress image file size |
| `image.convert` | Convert between image formats |
| `image.crop` | Crop image to region |
| `image.download` | Download image from URL |
| `image.ocr` | Extract text from image (OCR) |
| `image.qrcode_generate` | Generate QR code image |
| `image.resize` | Resize image dimensions |
| `image.rotate` | Rotate image by degrees |
| `image.watermark` | Add watermark to image |
