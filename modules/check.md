# Check

Type-checking modules for runtime value inspection. Use these to validate data types and make branching decisions in workflows.

## Modules

| Module | Description |
|--------|-------------|
| `check.is_array` | Check if value is an array |
| `check.is_empty` | Check if value is empty |
| `check.is_null` | Check if value is null |
| `check.is_number` | Check if value is a number |
| `check.is_object` | Check if value is an object |
| `check.is_string` | Check if value is a string |
| `check.type_of` | Get the type of a value |
