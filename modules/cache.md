# Cache

In-memory key-value cache with optional TTL (time-to-live) support. Use these modules to store and retrieve temporary data between workflow steps.

## Modules

| Module | Description |
|--------|-------------|
| `cache.clear` | Clear cache entries |
| `cache.delete` | Delete cache entry |
| `cache.get` | Get value from cache |
| `cache.set` | Set value with optional TTL |
