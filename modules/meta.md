# Meta

Modules for managing flyto-core modules themselves -- listing, generating, testing, and updating module documentation.

## Modules

| Module | Description |
|--------|-------------|
| `meta.modules.generate` | Generate a new module scaffold |
| `meta.modules.list` | List all available modules |
| `meta.modules.test_generator` | Generate tests for a module |
| `meta.modules.update_docs` | Update module documentation |
