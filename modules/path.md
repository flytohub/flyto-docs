# Path

Modules for file path manipulation -- join, split, normalize, and inspect path components.

## Modules

| Module | Description |
|--------|-------------|
| `path.basename` | Get filename from path |
| `path.dirname` | Get directory from path |
| `path.extension` | Get file extension |
| `path.is_absolute` | Check if path is absolute |
| `path.join` | Join path segments |
| `path.normalize` | Normalize path (resolve `.` and `..`) |
