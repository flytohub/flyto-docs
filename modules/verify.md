# Verify

Visual verification and design comparison modules. Compare screenshots, validate against Figma designs, and generate verification reports.

## Modules

| Module | Description |
|--------|-------------|
| `verify.annotate` | Annotate screenshot with markers |
| `verify.capture` | Capture element or page screenshot for verification |
| `verify.compare` | Compare two screenshots |
| `verify.figma` | Compare screenshot against Figma design |
| `verify.report` | Generate verification report |
| `verify.ruleset` | Define verification rules |
| `verify.run` | Run verification suite |
| `verify.spec` | Define verification specification |
| `verify.visual_diff` | Generate visual diff between images |
