# Data Transform

Modules for parsing, converting, and transforming structured data. Covers CSV, JSON, XML, YAML formats and object manipulation utilities.

## CSV

| Module | Description |
|--------|-------------|
| `data.csv.read` | Read and parse CSV data |
| `data.csv.write` | Write data to CSV format |
| `data.json_to_csv` | Convert JSON data to CSV |

## JSON

| Module | Description |
|--------|-------------|
| `data.json.parse` | Parse JSON string to object |
| `data.json.stringify` | Stringify object to JSON |

## XML

| Module | Description |
|--------|-------------|
| `data.xml.generate` | Generate XML from object |
| `data.xml.parse` | Parse XML to object |

## YAML

| Module | Description |
|--------|-------------|
| `data.yaml.generate` | Generate YAML from object |
| `data.yaml.parse` | Parse YAML to object |

## Pipeline & Template

| Module | Description |
|--------|-------------|
| `data.pipeline` | Chain multiple data transformations |
| `data.text.template` | Render text templates with variables |

## Object Utilities

| Module | Description |
|--------|-------------|
| `object.keys` | Get object keys |
| `object.merge` | Merge objects |
| `object.omit` | Omit keys from object |
| `object.pick` | Pick keys from object |
| `object.values` | Get object values |
