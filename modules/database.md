# Database

Modules for interacting with databases. Supports MongoDB, MySQL, PostgreSQL, and Redis.

## Modules

| Module | Description |
|--------|-------------|
| `db.mongodb.find` | Query documents in MongoDB |
| `db.mongodb.insert` | Insert documents into MongoDB |
| `db.mysql.query` | Execute MySQL query |
| `db.postgresql.query` | Execute PostgreSQL query |
| `db.redis.get` | Get value from Redis |
| `db.redis.set` | Set value in Redis |
