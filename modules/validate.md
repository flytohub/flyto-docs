# Validate

Modules for validating common data formats -- emails, URLs, phone numbers, IPs, UUIDs, credit cards, and JSON schemas.

## Modules

| Module | Description |
|--------|-------------|
| `validate.credit_card` | Validate credit card number |
| `validate.email` | Validate email address format |
| `validate.ip` | Validate IP address |
| `validate.json_schema` | Validate data against JSON Schema |
| `validate.phone` | Validate phone number |
| `validate.url` | Validate URL format |
| `validate.uuid` | Validate UUID format |
