# Cloud Services

Modules for interacting with cloud storage providers and Google Workspace services.

## AWS S3

| Module | Description |
|--------|-------------|
| `aws.s3.delete` | Delete object from S3 |
| `aws.s3.download` | Download file from S3 |
| `aws.s3.list` | List objects in S3 bucket |
| `aws.s3.upload` | Upload file to S3 |
| `cloud.aws_s3.download` | Download file from AWS S3 (simplified) |
| `cloud.aws_s3.upload` | Upload file to AWS S3 (simplified) |

## Azure Blob Storage

| Module | Description |
|--------|-------------|
| `cloud.azure.download` | Download file from Azure Blob Storage |
| `cloud.azure.upload` | Upload file to Azure Blob Storage |

## Google Cloud Storage

| Module | Description |
|--------|-------------|
| `cloud.gcs.download` | Download file from Google Cloud Storage |
| `cloud.gcs.upload` | Upload file to Google Cloud Storage |

## Google Workspace

| Module | Description |
|--------|-------------|
| `google.calendar.create_event` | Create a Google Calendar event |
| `google.calendar.list_events` | List Google Calendar events |
| `google.gmail.search` | Search Gmail messages |
| `google.gmail.send` | Send email via Gmail |
