# Format

Modules for formatting values into human-readable strings. Handles currency, durations, file sizes, numbers, and percentages.

## Modules

| Module | Description |
|--------|-------------|
| `format.currency` | Format number as currency |
| `format.duration` | Format duration in human-readable form |
| `format.filesize` | Format bytes as file size |
| `format.number` | Format number with locale options |
| `format.percentage` | Format number as percentage |
