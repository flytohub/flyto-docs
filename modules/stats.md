# Stats

Statistical analysis modules for computing descriptive statistics on numeric datasets.

## Modules

| Module | Description |
|--------|-------------|
| `stats.mean` | Calculate arithmetic mean |
| `stats.median` | Calculate median value |
| `stats.min_max` | Find minimum and maximum values |
| `stats.mode` | Find most frequent value |
| `stats.percentile` | Calculate percentile value |
| `stats.std_dev` | Calculate standard deviation |
| `stats.sum` | Calculate sum |
| `stats.variance` | Calculate variance |
