# Docker

Modules for managing Docker containers and images. Build, run, inspect, and manage container lifecycle.

## Modules

| Module | Description |
|--------|-------------|
| `docker.build` | Build a Docker image |
| `docker.inspect_container` | Inspect container details |
| `docker.logs` | View container logs |
| `docker.ps` | List running containers |
| `docker.run` | Run a Docker container |
| `docker.stop` | Stop a running container |
