# Regex

Modules for working with regular expressions -- match, extract, replace, split, and test patterns.

## Modules

| Module | Description |
|--------|-------------|
| `regex.extract` | Extract matched groups from string |
| `regex.match` | Match string against pattern |
| `regex.replace` | Replace matches with substitution |
| `regex.split` | Split string by pattern |
| `regex.test` | Test if string matches pattern (boolean) |
