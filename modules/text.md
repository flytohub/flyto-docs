# Text

Modules for text analysis and extraction -- count words/characters, detect encoding, and extract structured data from text.

## Modules

| Module | Description |
|--------|-------------|
| `text.char_count` | Count characters in text |
| `text.detect_encoding` | Detect text encoding |
| `text.extract_emails` | Extract email addresses from text |
| `text.extract_numbers` | Extract numbers from text |
| `text.extract_urls` | Extract URLs from text |
| `text.word_count` | Count words in text |
