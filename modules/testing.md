# Testing

Assertion modules for validating values within workflows. For test execution modules (unit, E2E, BDD, security), see [Atomic](/modules/atomic).

## Modules

| Module | Description |
|--------|-------------|
| `test.assert_contains` | Assert value contains substring or element |
| `test.assert_equal` | Assert two values are equal |
| `test.assert_greater_than` | Assert value is greater than threshold |
| `test.assert_length` | Assert array/string length |
| `test.assert_not_null` | Assert value is not null |
| `test.assert_true` | Assert value is truthy |
