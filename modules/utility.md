# Utilities

General-purpose utility modules for date/time operations, delays, hashing, and random value generation.

## Date & Time

| Module | Description |
|--------|-------------|
| `datetime.add` | Add duration to datetime |
| `datetime.format` | Format datetime to string |
| `datetime.parse` | Parse datetime from string |
| `datetime.subtract` | Subtract duration from datetime |
| `utility.datetime.now` | Get current datetime |

## Miscellaneous

| Module | Description |
|--------|-------------|
| `utility.delay` | Delay execution |
| `utility.hash.md5` | Generate MD5 hash |
| `utility.random.number` | Generate random number |
| `utility.random.string` | Generate random string |
