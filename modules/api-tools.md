# API Tools

Modules for interacting with external APIs. Includes GitHub integration, HTTP request utilities, and search engine APIs.

## GitHub

| Module | Description |
|--------|-------------|
| `api.github.create_issue` | Create a new issue in a GitHub repository |
| `api.github.create_pr` | Create a new pull request in a GitHub repository |
| `api.github.get_repo` | Get information about a GitHub repository |
| `api.github.list_issues` | List issues from a GitHub repository |
| `api.github.list_repos` | List repositories for a GitHub user |

## HTTP & Search

| Module | Description |
|--------|-------------|
| `core.api.google_search` | Use Google Custom Search API |
| `core.api.http_get` | Send HTTP GET request |
| `core.api.http_post` | Send HTTP POST request |
| `core.api.serpapi_search` | Use SerpAPI to search keywords |
