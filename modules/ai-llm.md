# AI & LLM

Modules for interacting with large language models and AI services. Includes support for cloud providers (OpenAI, Anthropic, Google) and local models via Ollama, as well as autonomous agent patterns.

## Modules

| Module | Description |
|--------|-------------|
| `agent.autonomous` | Self-directed AI agent with memory and goal-oriented behavior |
| `agent.chain` | Sequential AI processing chain with multiple steps |
| `ai.local_ollama.chat` | Chat with local LLM via Ollama (completely offline) |
| `api.anthropic.chat` | Send a chat message to Anthropic Claude AI |
| `api.google_gemini.chat` | Send a chat message to Google Gemini AI |
| `api.openai.chat` | Send a chat message to OpenAI GPT models |
| `api.openai.image` | Generate images using DALL-E |
