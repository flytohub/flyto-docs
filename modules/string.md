# String Operations

Modules for string manipulation -- case conversion, splitting, padding, templating, and more.

## Modules

| Module | Description |
|--------|-------------|
| `string.lowercase` | Convert to lowercase |
| `string.pad` | Pad string to length |
| `string.replace` | Replace substring |
| `string.reverse` | Reverse string |
| `string.slugify` | Convert to URL-friendly slug |
| `string.split` | Split string by delimiter |
| `string.template` | Render string template with variables |
| `string.titlecase` | Convert to title case |
| `string.trim` | Trim whitespace |
| `string.truncate` | Truncate to max length |
| `string.uppercase` | Convert to uppercase |
