# Encode / Decode

Modules for encoding and decoding data in various formats including Base64, hex, URL, and HTML entities.

## Modules

| Module | Description |
|--------|-------------|
| `decode.base64` | Decode Base64 string |
| `decode.hex` | Decode hex string |
| `decode.url` | Decode URL-encoded string |
| `encode.base64` | Encode to Base64 |
| `encode.hex` | Encode to hex |
| `encode.html` | Encode HTML entities |
| `encode.url` | URL-encode string |
