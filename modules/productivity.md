# Productivity

Modules for integrating with productivity platforms and payment services.

## Google Sheets

| Module | Description |
|--------|-------------|
| `api.google_sheets.read` | Read data from Google Sheets |
| `api.google_sheets.write` | Write data to Google Sheets |

## Notion

| Module | Description |
|--------|-------------|
| `api.notion.create_page` | Create a Notion page |
| `api.notion.query_database` | Query a Notion database |

## Airtable

| Module | Description |
|--------|-------------|
| `productivity.airtable.create` | Create Airtable record |
| `productivity.airtable.read` | Read Airtable records |
| `productivity.airtable.update` | Update Airtable record |

## Stripe (Payments)

| Module | Description |
|--------|-------------|
| `payment.stripe.create_payment` | Create a Stripe payment |
| `payment.stripe.get_customer` | Get Stripe customer details |
| `payment.stripe.list_charges` | List Stripe charges |
