# Analysis

Modules for analyzing HTML content. Extract metadata, forms, tables, and structural information from web pages and HTML documents.

## Modules

| Module | Description |
|--------|-------------|
| `analysis.html.analyze_readability` | Analyze content readability |
| `analysis.html.extract_forms` | Extract form data from HTML |
| `analysis.html.extract_metadata` | Extract metadata from HTML |
| `analysis.html.extract_tables` | Extract table data from HTML |
| `analysis.html.find_patterns` | Find repeating data patterns in HTML |
| `analysis.html.structure` | Analyze HTML DOM structure |
