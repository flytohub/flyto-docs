# Archive

Modules for creating and extracting compressed archives. Supports ZIP, TAR, and gzip formats.

## Modules

| Module | Description |
|--------|-------------|
| `archive.gunzip` | Decompress a gzip-compressed file |
| `archive.gzip` | Compress a single file using gzip |
| `archive.tar_create` | Create a TAR archive |
| `archive.tar_extract` | Extract files from a TAR archive |
| `archive.zip_create` | Create a ZIP archive |
| `archive.zip_extract` | Extract files from a ZIP archive |
