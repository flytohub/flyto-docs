# Crypto

Cryptographic modules for encryption, decryption, hashing, JWT handling, and secure random generation.

## Modules

| Module | Description |
|--------|-------------|
| `crypto.decrypt` | AES decryption |
| `crypto.encrypt` | AES encryption |
| `crypto.hmac` | Generate HMAC |
| `crypto.jwt_create` | Create JWT tokens |
| `crypto.jwt_verify` | Verify JWT tokens |
| `crypto.random_bytes` | Secure random bytes |
| `crypto.random_string` | Secure random string |
