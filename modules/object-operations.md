# Object Operations

Modules for advanced object manipulation -- deep merge, flatten/unflatten nested structures, and dot-path access.

## Modules

| Module | Description |
|--------|-------------|
| `object.deep_merge` | Deep merge multiple objects |
| `object.flatten` | Flatten nested object to dot-notation keys |
| `object.get` | Get value by dot-path |
| `object.set` | Set value by dot-path |
| `object.unflatten` | Unflatten dot-notation keys to nested object |
