# Convert

Type conversion modules for casting values between data types within workflows.

## Modules

| Module | Description |
|--------|-------------|
| `convert.to_array` | Convert value to array |
| `convert.to_boolean` | Convert value to boolean |
| `convert.to_number` | Convert value to number |
| `convert.to_object` | Convert value to object |
| `convert.to_string` | Convert value to string |
